Class: QueueManager

A singleton class responsible for managing inter-thread communication via queues and events.

Class Attributes

  • _instance – Stores the singleton instance.
  • _lock – A threading lock to ensure thread-safe singleton initialization.
  • status_queue – A queue for tracking status updates.
  • input_queue – A queue for handling input data.
  • response_queue – A queue for storing responses.
  • completion_event – A threading event to signal when an operation is complete.
  • _sheetname – A shared string resource with a lock.
  • _sheetname_lock – A lock to ensure safe access to _sheetname.

Constructor: __new__(cls)

Implements a thread-safe singleton pattern to ensure only one instance of QueueManager is created.

Behavior:

  • Uses a lock to ensure only one instance is initialized.
  • Creates and initializes the queues and event.
  • Ensures safe access to the shared _sheetname variable.
