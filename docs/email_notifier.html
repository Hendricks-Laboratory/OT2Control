EmailNotifier Documentation

Author: Cold Hard Cache

Description: The EmailNotifier class is responsible for sending email notifications when a robot reaction process is complete. It utilizes the Gmail API for sending emails and listens for an event triggered by the robot controller.

Class: EmailNotifier

Imports:

  • os – Handles file paths.
  • base64 – Encodes email messages for the Gmail API.
  • email.mime.text.MIMEText – Constructs email messages.
  • google.auth.transport.requests.Request – Handles authentication requests.
  • google_auth_oauthlib.flow.InstalledAppFlow – Manages OAuth 2.0 authentication.
  • googleapiclient.discovery.build – Builds the Gmail API service.
  • threadManager.QueueManager – Manages event queues for completion detection.
  • threadManager.ThreadManager – Manages threading (not explicitly used in this class).

Class Attributes

  • SCOPES – Defines the required OAuth 2.0 permissions for sending emails.

Constructor: __init__(self, recipient)

Initializes the email notifier and sets up authentication with the Gmail API.

Parameters:

  • recipient (str) – The email address of the recipient.

Functionality:

  • Retrieves the completion event from QueueManager.
  • Authenticates the user with the Gmail API.
  • Initializes the Gmail service.

Methods

get_credentials(self)

Handles OAuth 2.0 authentication to retrieve credentials for accessing the Gmail API.

Returns:

  • Credentials object if authentication is successful.
  • None if authentication fails.

Behavior:

  • Loads credentials from a JSON file.
  • Prompts the user to authenticate via a local web server.

create_email(self, sender, to, subject, body)

Creates an email message in MIME format and encodes it for Gmail API transmission.

Parameters:

  • sender (str) – The sender's email address.
  • to (str) – The recipient's email address.
  • subject (str) – The email subject.
  • body (str) – The email content.

Returns:

  • A dictionary containing the encoded email message.

send_email(self)

Sends an email notification via the Gmail API.

Behavior:

  • Creates an email using create_email.
  • Sends the email using the Gmail API.
  • Prints a success message if the email is sent successfully.
  • Prints an error message if sending fails.

watch_event(self)

Waits for the completion event and sends an email notification when triggered.

Behavior:

  • Listens for the completion event.
  • Calls send_email() when the event is triggered.
  • Resets the event after sending the email.

Usage Example

To use the EmailNotifier class, instantiate it with a recipient email address and call watch_event():


from email_notifier import EmailNotifier

notifier = EmailNotifier("user@example.com")
notifier.watch_event()

Dependencies

Ensure you have the following installed before running the script:


pip install google-auth google-auth-oauthlib google-auth-httplib2 google-api-python-client

Notes

  • The script requires a valid credentials.json file for Gmail API authentication.
  • OAuth authentication may require user interaction via a browser.
  • The email sender must have permission to use the Gmail API.
